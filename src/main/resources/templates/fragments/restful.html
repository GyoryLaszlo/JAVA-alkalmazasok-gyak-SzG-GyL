RESTful API

Üzenetek kezelése RESTful API-n keresztül.

Elérhető végpontok

  • GET /api/messages – Összes üzenet lekérése
  • GET /api/messages/{id} – Egy üzenet lekérése
  • POST /api/messages – Új üzenet mentése
  • PUT /api/messages/{id} – Üzenet módosítása
  • DELETE /api/messages/{id} – Üzenet törlése
Megjegyzés:

Formátum: Az API JSON formátumban adja vissza az adatokat.

Tesztelés: Postman vagy cURL segítségével.
